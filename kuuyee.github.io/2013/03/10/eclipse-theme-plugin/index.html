Eclipse配色方案插件

kuuyee  |  2013-03-10  |  Eclipse   IDE   Theme  

最近发现了一个Eclipse配色方案插件，这回给Eclipse配色太方便了。

插件主页：http://eclipsecolorthemes.org/

插件提供了上万种配色方案，非常漂亮，上图看下：

安装很简单，打开 Eclipse Marketplace

查找 Eclipse Color Theme，会找到这个插件。

安装完成后进入 Preferences —> Appearance —> Color Theme 就可选择主题了
